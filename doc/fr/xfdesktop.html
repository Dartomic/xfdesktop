Introduction

Cette application xfdesktop gère le fond du bureau. Elle installe une image et/ou une couleur de fond et fournit un menu popup quand vous faites un clic droit ou milieu avec la souris. À côté de l'application, une fenêtre de dialogue est installée pour être utilisée dans le gestionnaire de paramètres Xfce 4 : cette fenêtre vous permet de changer l'image du fond et la couleur du bureau.

Fond d'écran

Vous pouvez modifier le fond d'écran en cliquant sur le bouton "Bureau : fond d'écran" dans le gestionnaire de paramètres.

Figure 1. Paramètres du fond d'écran

Couleur

Cliquer sur le bouton "Bureau : fond d'écran" et une nouvelle fenêtre de nouveau dialogue apparaît :

Figure 2. Sélection de la couleur de fond

Ce dialogue vous permet de choisir une couleur pour le fond d'écran. Cette couleur remplit tout l'écran et demeure visible à travers une image transparente. Il y a plusieurs manières de faire cela :

  • Vous pouvez utiliser l'anneau des couleurs pour choisir la couleur souhaitée, puis le triangle interne pour ajuster la luminosité de cette couleur. Les valeurs correspondantes de luminosité, saturation et les valeurs de rouge, vert, bleu s'affichent dans les zones de droite.
  • Vous pouvez utiliser la pipette et, en cliquant dessus, copier une couleur de votre écran.
  • Vous pouvez saisir directement les valeurs, type HTML, en héxadécimal ou en tapant le nom de la couleur dans la zone "Nom de la couleur".

Choisissez "Utiliser uniquement la couleur" si vous voulez que xfdesktop ignore l'image de fond et n'utilise que la couleur choisie.

Image

L'entrée "fichier" montre le chemin de l'image actuellement sélectionnée. Si vous souhaitez la changer, déposez simplement une image du gestionnaire de fichier vers la zone de dialogue, modifiez le chemin ou cliquez sur le bouton "Parcourir", et naviguez dans le système de fichiers.

Trois options d'affichage (mosaïque, étiré et centré) peuvent être utilisées pour couvir le fond d'écran avec l'image selon sa taille. Vous pouvez aussi choisir "automatique" si vous préférez laisser xfdesktop s'occuper de cela.

Vous pouvez créer une liste d'images de fond qui seront utilisées au hasard au démarrage. Cliquez sur "Nouvelle liste" pour créer cette liste :

Figure 3. Liste de fonds d'écran

Vous pouvez désirer modifier le nom et chemin par défaut de votre liste de fonds dans la zone "Fichier liste".

Vous pouvez facilement ajouter une image en cliquant sur le bouton "plus". Cette action ouvre une fenêtre pour naviguer dans votre système. Vous pouvez retirer un fichier en le sélectionnant dans la liste et en cliquant sur le bouton "moins". Quand votre liste est complète, cliquez sur le bouton "sauver". Si vous souhaitez la modifier par la suite, utilisez le bouton "Edition de la liste" du gestionnaire de fond d'écran.

Vous pouver récupérer une ancienne liste en déposant son nom, à partir d'un gestionnaire de fichier, dans la fenêtre du gestionnaire de fonds d'écran.

Pour actualiser le fond d'écran avec une nouvelle image, prise au hasard dans la liste, relancer xfdesktop, à partir d'un terminal ou à partir de la fenêtre exécuter, ou xfdesktop [-reload].

Espaces de travail

Vous pouvez lancer le gestionnaire de paramètres de l'espace de travail en cliquant sur "Bureau : espaces virtuels" dans le gestionnaire global.

Figure 4. Paramètres d'espaces virtuels

Espaces virtuels

Cette option vous permet de choisir le nombre d'espaces virtuels que vous souhaitez utiliser (entre 1 et 32).

Nom des espaces virtuels

Ce champ contient la liste des espaces disponibles. Si vous modifier le nom de l'un d'eux, cliquez sur son numéro et remplissez la zone de saisie qui s'ouvre. Ensuite, cliquez sur le bouton "Appliquer" ou simplement sur la touche Entrée, pour revenir à la liste.

xfdesktop menus du bureau

xfdesktop fournit un menu qui apparaît lorsque vous cliquez sur le fond du bureau avec, soit le bouton droit, soit le bouton du milieu de la souris.

Un clic droit sur le fond du bureau ouvre un menu qui vous permet de lancer des applications. Son fichier de configuration, menu.xml, est placé dans $sysconfdir/xfce4/menu.xml. $sysconfdir est souvent /etc pour les paquetages compilés et /usr/local/etc pour les sources.

Actuellement, le seul moyen de modifier ce menu est de le faire manuellement. On peut espérer qu'une future version version de XFce 4 possèdera un éditeur graphique. Pour l'instant, vous devez copier le fichier menu.xml vers ~/.xfce4/menu.xml et l'éditer. Ce n'est pas difficile. Jetez un coup d'oeil sur le fichier et les exemples commentés. Être un peu habitué au format xml ou html peut aider. Le champ icône n'a aucun effet, son traitement n'étant pas encore implanté.

NOTE : le fichier menu.xml doit être codé en utf-8 pour pouvoir afficher les caractères accentués ou autres caractères non-ASCII. Donc vous avez besoin d'un éditeur qui permet l'utf-8, tel (NdT : entre autres) gedit.

Figure 5. xfdesktop menu clic droit

Cliquer avec le bouton du milieu affiche la liste des applications ouvertes, classées par ordre des espaces de travail. Cela vous permet aussi d'ajouter ou de supprimer un espace virtuel.

Figure 6. xfdesktop menu clic du milieu

Vous pouvez aussi ouvrir le menu ou la liste des fenêtres depuis une ligne de commande. Cela peut être utile pour créer un raccourci clavier. Pour ouvrir le menu, lancez la commande xfdesktop [-menu] et pour la liste des fenêtres utilisez xfdesktop [-windowlist].
