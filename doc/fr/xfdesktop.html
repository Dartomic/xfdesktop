Introduction

L'application xfdesktop gère le fond du bureau. Elle installe une image et/ou une couleur de fond et fournit un menu popup ou une liste des applications ouvertes quand vous cliquez respectivement avec le bouton droit ou central de la souris.

Le module xfdesktop installe une fenêtre de dialogue de paramètrage utilisant le gestionnaire de paramètres Xfce 4. Dans cette fenêtre, vous pouvez changer l'image de fond et la couleur du bureau ; ainsi que les paramètres du menu et de la liste des bureaux.

Fond d'écran

Vous pouvez lancer le dialogue de gestion des paramètres du bureau en cliquant sur le bouton "Bureau" dans le gestionnaire de paramètres Xfce.

Figure 1. Paramètres de fond d'écran

Couleur

Ce dialogue vous permet de choisir un style de couleur pour le fond d'écran : couleur unie, dégradé horizontal ou vertical. Selon votre choix, vous pouvez sélectionner une ou deux couleurs avec les boutons "première couleur" et "seconde couleur". Ces boutons ouvrent une fenêtre de dialogue de choix de couleurs :

Figure 2. Sélection de la couleur de fond

 

  • Vous pouvez utiliser l'anneau des couleurs avec un triangle interne : sélectionnez la couleur souhaitée dans l'anneau externe puis utilisez le triangle interne pour ajuster la luminosité de cette couleur. Les valeurs correspondantes de luminosité, saturation et les valeurs de rouge, vert, bleu s'affichent dans les zones de droite de la fenêtre.
  • Vous pouvez utiliser la pipette en cliquant dessus, puis cliquez n'importe où dans votre écran pour copier une couleur.
  • Vous pouvez saisir directement les valeurs de la couleur, type HTML, en héxadécimal ou en tapant simplement le nom de la couleur dans la zone "Nom de la couleur".

Les couleurs sélectionnées rempliront entièrement l'écran et resteront visible à travers une image de fond transparente.

Image

Décochez le choix "Montrer l'image" si vous voulez que xfdesktop ignore l'image de fond et n'utilise que les couleurs choisies.

L'entrée "fichier" montre le chemin de l'image actuellement sélectionnée. Si vous souhaitez la changer, déposez simplement une image sur la zone de saisie du chemin, modifiez le chemin ou cliquez sur le bouton "Parcourir", et utilisez la fenêtre de recherche habituelle du système de fichiers.

Vous pouvez créer une liste d'images de fond qui seront utilisées au hasard au démarrage. Cliquez sur "Nouvelle liste" pour ouvrir la fenêtre de dialogue de création de cette liste :

Figure 3. Liste de fonds d'écran

Vous pouvez modifier le nom et le chemin par défaut de votre liste de fonds d'écran dans la zone "Fichier liste".

Vous pouvez facilement ajouter une image en cliquant sur le bouton "plus". Cette action ouvre une fenêtre pour naviguer dans votre système. Vous pouvez retirer un fichier en le sélectionnant dans la liste et en cliquant sur le bouton "moins". Quand votre liste est complète, cliquez sur le bouton "sauver". Si vous souhaitez la modifier par la suite, utilisez le bouton "Edition de la liste" du gestionnaire de fond d'écran.

Vous pouver réutiliser une ancienne liste en déposant son nom, à partir d'un gestionnaire de fichier, dans la fenêtre du gestionnaire de fonds d'écran.

Pour actualiser le fond d'écran avec une nouvelle image, prise au hasard dans la liste, relancer simplement xfdesktop, à partir d'un terminal ou à partir de la fenêtre exécuter, ou en tapant xfdesktop [-reload].

Quatre options de style (Mosaïque, À l'échelle, Centré, Étiré) permettent de préciser l'ajustement de l'image de fond à l'écran selon sa taille. Vous pouvez aussi choisir "Auto" si vous préférez laisser xfdesktop gérer ce paramètre.

Le curseur au bas de la fenêtre vous permet d'ajuster la brillance de l'image et/ou sa couleur.

xfdesktop reconnait xinerama et les modes multi-écrans. En mode multi-écrans, le paramétrage "Fonds d'écran" propose un onglet par écran.

xfdesktop menu du bureau

xfdesktop fournit un menu et une liste des fenêtres qui apparaît lorsque vous cliquez sur le fond du bureau avec, soit le bouton droit, soit le bouton du milieu de la souris.

L'onglet Menu de la fenêtre de dialogue des paramètres du bureau vous permet de configurer le comportement du menu.

Figure 4. Onglet Menu

Si vous souhaitez utiliser le menu de bureau et/ou la liste des fenêtres les options correspondantes doivent être sélectionnées dans la fenêtre des paramètres.

Un clic droit sur le bureau ouvre un menu qui vous permet de lancer des applications. Son fichier de configuration, menu.xml, est placé dans le chemin $sysconfdir/xdg/xfce4/menu.xml. $sysconfdir est, par défaut, souvent /etc pour les paquetages binaires et /usr/local/etc pour les sources.

Quoiqu'il soit possible de le faire manuellement, la méthode recommandée pour éditer le fichier menu.xml est d'utiliser l'éditeur de menu de Xfce 4 qui peut être lancé par la commande xfce4-menueditor ou en utilisant le bouton "Édition du menu de bureau" disponible dans l'onglet Menu de la fenêtre de dialogue. L'éditeur de menu accepte aussi le glisser-déposer du gestionnaire de fichiers.

Si vous avez modifié le menu via xfce4-menueditor, le fichier de vos préférences pour le menu est sauvegardé sous $XDG_CONFIG_HOME/xfce4/desktop/menu.xml. $XDG_CONFIG_HOME est par défaut ~/.config. Si vous modifiez ce fichier manuellement, le copier d'abord à cet endroit est préférable.

Figure 5. xfdesktop éditeur de menu

NOTE : si vous choisissez de le modifier manuellement, le fichier menu.xml doit être codé en UTF-8 pour pouvoir afficher les caractères accentués ou autres caractères non-ASCII. Donc vous avez besoin d'un éditeur qui permet l'UTF-8, tel gedit.

Figure 6. xfdesktop menu clic droit

Cliquer avec le bouton du milieu sur le bureau affiche la liste des applications ouvertes, classées par ordre des espaces de travail. Cela vous permet aussi d'ajouter ou de supprimer un espace virtuel.

Figure 7. xfdesktop menu clic du milieu

Pour diminuer le niveau matériel requis pour utiliser xfdesktop, vous pouvez supprimer l'utilisation des icônes dans le menu de bureau et dans la liste des fenêtres depuis la fenêtre de paramètrage.

Vous pouvez aussi ouvrir le menu ou la liste des fenêtres depuis la ligne de commande. Cela peut être utile pour créer un raccourci clavier. Pour ouvrir le menu, lancez la commande xfdesktop -menu et pour la liste des fenêtres utilisez xfdesktop -windowlist.
