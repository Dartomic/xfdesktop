Introduction

The xfdesktop application manages the desktop background. It sets a background image and/or color and it pops up a menu when you click on it with your mouse. Along with the apllication two settings dialogs are installed that can be used with the XFce 4 Settings Manager. These dialogs allow you to change the background image and color and manage the number of workspaces or virtual desktops.
